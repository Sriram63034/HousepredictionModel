🏠 House Price Prediction using Linear Regression (Single Feature)
📌 Project Overview

This project implements a Simple Linear Regression model to predict house prices based on one feature: house size (square feet).

The goal of this project was to understand the basic concepts of Machine Learning, including:

Supervised Learning

Linear Regression

Training a model

Making predictions

Evaluating model performance

This is my first Machine Learning project.

📊 Dataset

A housing dataset was used where the price of the house depends on its size.

Feature

Size – House size in square feet

Target

Price – House price

The model learns the relationship between house size and house price.

Example relationship:

Price = m × Size + b

Where:

m → slope (weight)

b → intercept

🛠 Technologies Used

Python

NumPy

Scikit-Learn

Matplotlib (for visualization)

⚙️ Machine Learning Workflow

Load dataset

Split dataset into training and testing data

Train a Linear Regression model

Predict house prices using the trained model

Evaluate the model using Mean Squared Error (MSE)

Allow the user to input house size to predict price

📈 Example Prediction

User Input:

Enter house size: 2000 sq ft

Output:

Estimated Price: ₹45.00 lakhs
